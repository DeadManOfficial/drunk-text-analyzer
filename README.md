## Features

| Feature | Description |
|---------|-------------|
| **Regret Score** | 0-100 scale of how bad it is |
| **Roast Level** | How much your friends will laugh |
| **Sober Translation** | What you actually meant |
| **Recovery Plan** | Steps to fix the damage |
| **Vibe Check** | Category of drunk text |

---

## Quick Start

```bash
git clone https://github.com/DeadManOfficial/drunk-text-analyzer.git
cd drunk-text-analyzer
npm install
npm run dev
```

Open `http://localhost:3000`

---

## How It Works

1. Paste your drunk text
2. AI analyzes the damage
3. Get your regret score
4. Share your shame (optional)
